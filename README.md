## Run Locally

Clone the project

```bash
  git clone https://github.com/yogasatriautama/Sanber52-Cypress-Kelompok7.git
```

Go to the project directory

```bash
  cd Sanber52-Cypress-Kelompok7
```

Install dependencies

```bash
  npm install
```

Run cypress

```bash
  npx cypress open
```
